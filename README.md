## About

News feed page made with: `HTML`, `CSS`, `SCSS` and `JS`.

### Development requirements

- Node.js
- Npm

### Instructions

#### Develop

- Install dependencies: 

```bash
npm install
```

- Run project:

  - For compile SCSS and watch changes:

    ```bash
    npm run scss
    ```

  - Build SCSS for production (`Minify`):

    ```bash
    npm run build-scss
    ```

#### Run

- Just open `./index.html` in the web brwoser.
